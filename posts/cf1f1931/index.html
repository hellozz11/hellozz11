2. Code-Auswertung

  • OWASP-Abhängigkeitsprüfung
    Dependency-Check ist ein Software Composition Analysis (SCA) -Tool, mit dem öffentlich gemeldete Sicherheitslücken in Projektabhängigkeiten erkannt werden. Dazu ermittelt es, ob eine bestimmte Abhängigkeit eine CPE-Kennung (Common Platform Enumeration) aufweist. Wenn dies gefunden wird, wird ein Bericht generiert, der auf den relevanten CVE-Eintrag verweist.

https://github.com/jeremylong/DependencyCheck

  • Cougar-Scan
    Puma Scan ist ein .NET-Software-Sicherheitscode-Analysetool, das eine kontinuierliche Quellcodeanalyse in Echtzeit bietet, während das Entwicklungsteam Code schreibt. In Visual Studio werden Sicherheitslücken in der Entwicklungsumgebung sofort als Rechtschreibprüfung und Compiler-Warnungen angezeigt, um zu verhindern, dass Sicherheitsverletzungen in Ihre Anwendung gelangen. Puma Scan ist ebenfalls in den Build integriert, um eine Sicherheitsanalyse zur Kompilierungszeit zu ermöglichen.

https://github.com/pumasecurity/puma-scan

6. Hardware / Eingebettet

  • ChipWhisperer
    ChipWhisperer ist eine Open-Source-Werkzeugkette, die sich der Hardware-Sicherheitsforschung widmet. Die Toolchain besteht aus mehreren Schichten von Open-Source-Komponenten:
      1. Hardware: ChipWhisperer verwendet ein Capture Board und ein Target Board. Das Schaltplan- und PCB-Layout des ChipWhisperer-Lite Capture Boards und vieler Target Boards wird kostenlos zur Verfügung gestellt.
      1. Firmware: Auf der ChipWhisperer-Hardware werden drei separate Firmwares verwendet. Das Capture Board verfügt über einen USB-Controller (in C), einen FPGA für High-Speed-Capture (in Verilog) und Open-Source-Firmware. Zusätzlich verfügt das Zielgerät über eine eigene Firmware, die viele Firmware-Beispiele für verschiedene Ziele enthält.
      1. Software: Die Software ChipWhisperer besteht aus einem Capture-Programm zur Steuerung der Hardware und einem Analyseprogramm zur Verarbeitung der erfassten Daten.

https://github.com/newaetech/chipwhisperer

  • JTAGulator: Entdecken Sie die Achillesferse der Hardware-Sicherheit

https://github.com/grandideastudio/jtagulator

  • Micro-Renovator: Bringen Sie die Prozessor-Firmware in den Code ein
    MicroRenovator bietet einen Mechanismus zum Bereitstellen von Prozessormikrocode unabhängig von den vom Hersteller und vom Betriebssystem bereitgestellten Aktualisierungen, indem ein benutzerdefiniertes EFI-Startskript hinzugefügt wird, das den Mikrocode vor dem Ausführen des Betriebssystems lädt. Dadurch wird die Ghost-Schwachstelle behoben, indem der Betriebssystemkernel in die Lage versetzt wird, den aktualisierten Mikrocode zu erkennen und zu aktivieren.

https://github.com/syncsrc/MicroRenovator

  • TumbleRF: RF Blur wird einfach
    TumbleRF ist ein fuzzing Framework zum Erstellen von Anwendungen für RF-Fuzz-Tests. TumbleRF zielt darauf ab, RF-Fuzzing zu implementieren, indem APIs bereitgestellt werden, um diese Technologien über Protokolle, Funkgeräte und Treiber hinweg zu vereinheitlichen.

https://github.com/riverloopsec/tumblerf

  • Walross: Machen Sie das Beste aus Ihrem Karten-Klongerät
    Walrus ist eine Android-App für kontaktlose Kartenklongeräte wie Proxmark3 und Chameleon Mini. Über eine einfache Google Pay-Style-Oberfläche können Sie Zugangskontrollkarten in Ihre Brieftasche lesen, um sie später zu schreiben oder zu simulieren.
    Während der Teilnahme an Red Team wurde Walrus für Evaluatoren der physischen Sicherheit konzipiert und unterstützt grundlegende Aufgaben wie das Lesen, Schreiben und Emulieren von Karten sowie gerätespezifische Funktionen wie Antennenabstimmung und Gerätekonfiguration. [https://github.com/TeamWalrus/Walrus] (https://github.com/TeamWalrus/Walrus)

    WASP OWTF ist ein Schwerpunkt auf Penetration und Sicherheitstests, OWASP Testing (v3 und v4), OWASP Top 10, PTES und Sicherheitsstandards wie NIST Web-Test-Framework. Das Tool ist in hohem Maße konfigurierbar, kann jeder leicht eine einfache Plug-in-Konfigurationsdatei erstellen oder einen neuen Test, ohne Entwicklung Erfahrung hinzufügen.

Https://github.com/owtf/owtf

  • OWASP JoomScan-Projekt
    OWASP Joomla ist ein Open Source Vulnerability Scanner entwickelt, um Auto-Sicherheitsanfälligkeit Joomla CMS Implementierung und Zuverlässigkeit Sicherungsaufgaben zu erfassen. Es kann nicht nur bekannte Angriff Schwachstellen erkennen, aber auch viele Fehler Konfiguration und Administrator-Defekte erkennen kann. Darüber hinaus bietet OWASP JoomScan eine benutzerfreundliche Oberfläche und Formatieren von Text und HTML-Ausgabe berichtet.

Https://github.com/rezasp/joomscan

  • WSSAT
    SSAT ist eine Open-Source-Web-Services-Sicherheit Scan-Tool, das durch Bearbeitung einer Konfigurationsdatei hinzufügen, aktualisieren oder löschen Verwundbarkeit eine dynamische Umgebung zur Verfügung stellt. Dieses Tool nimmt als Eingabe eine Adressliste WSDL-Dateien für Sicherheitslücken statische und dynamische Tests für jeden Dienst.

Https://github.com/YalcinYolalan/WSSAT
